1. JavaScript 概述

1.1 JavaScript 是什么

JavaScript 是一种运行在浏览器中的脚本语言，通常嵌入在 HTML 页面中，用来给网页添加交互行为。

常见作用：

  • 操作网页内容
  • 响应用户点击、输入、提交等事件
  • 向服务器发送请求
  • 修改页面样式和结构
  • 实现前端数据校验、动态渲染、异步加载

JavaScript 代码由浏览器中的 JavaScript 引擎解释并执行，通常不需要像 C、Java 那样提前编译。

1.3 JavaScript 的特点

  • 脚本语言：可以使用文本编辑器编写。
  • 解释执行：浏览器读取代码后逐行解释执行。
  • 基于对象：内置了大量对象，如 String、Array、Date、Math。
  • 弱类型：变量本身没有固定类型，变量保存的值才有类型。
  • 事件驱动：常通过点击、输入、加载等事件触发代码执行。

1.4 浏览器引擎

浏览器内部常见两类引擎：

  • 渲染引擎：解析 HTML/CSS，决定页面怎么显示。
  • JavaScript 引擎：解析并执行 JavaScript 代码。

常见浏览器与引擎：
