命令 用途
calc 弹计算器
chcp 65001 在Windows控制台中将字符编码设置为UTF-8

ipconfig
ipconfig /all 获取域名、IP地址、DHCP服务器、网关、MAC地址、主机名
net config Workstation 查看当前计算机的名称，计算机所属的域或工作组， 登录域，其他网络设置
net localgroup administrators 查看本地管理员组当中的用户成员
net localgroup administrators /add 将用户加入管理员组
net user 列出所有本地用户
net user 查看用户的详细信息

net view 查看局域网内其他主机名
net view /domain 查看有几个域
ping Ping命令是一个用于测试网络连接的工具。它通过发送ICMP回显请求消息到目标地址，并报告是否收到预期的ICMP回显应答
powershell.exe 启用powershell
powershell.exe -c 启用powershell执行命令
powershell.exe -nop 启用powershell时不加载环境配置
powershell.exe -w hidden 启用powershell时窗口隐藏

whoami /priv 查看当前用户及权限

Windows服务

Kerberos Key Distribution Center

Kerberos Database: 包含了一个 Realm 中所有的 principal、密码与其他信息。（默认：Berkeley DB）

Authentication Service(身份验证服务，AS): 进行用户信息认证，为客户端提供 Ticket Granting Tickets(票据，TGT)。

Ticket Granting Service(票据授予服务TGS): 验证 TGT 与 Authenticator，为客户端提供Service Tickets

活动目录

D（Domain，域）

活动目录的一种实现形式，主要由域控制器和成员计算机组成

DC（Domain Controller，域控制器）

域控制器是安装了活动目录的一台计算机（一般为Windows server），活动目录的数据都存储在域控制器内

一个域可以有多台域控制器

PDC（Primary Domain Controller，主域控制器）

在 AD域中，是指在一个域中主要负责处理账号登录请求和维护域中账号数据库的服务器。
