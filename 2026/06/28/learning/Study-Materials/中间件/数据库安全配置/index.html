Language），对数据库，表，列等进行创建，删除，更改 #### 5.1.1数据库操作 Show databases：查看当前MySQL服务器中有哪些数据库 Create databases 数据库名：创建新的数据库 Use 数据库名：切换数据库 Select database（）：查看当前正在使用的数据库 Drop database 数据库名：删除数据库 #### 5.1.2表操作 Show tables：查看当前数据库有哪些表 Desc 表名：查询表结构 Join：用来把多个表横向连接起来 select * from user join flag on user.id=flag.id 根据表中字段连接 Using和on是一样的，using是简化版的on Using(id)会将id列合并 ### 5.2DML语句 数据操纵语句（Data Manipulation Language），用于添加（增），删除（删），更新（改），查询（查）数据库 #### 5.2.1Select查询语句 基本格式： Select 列名 from 表名 查询多个列： Select 列名1，列名2，...，from 表名 查询所有列： Select * from 表名 使用where语句进行查询： Select 列名 from 表明 where 列名=xx Like主要用于where语句中搜索指定的匹配模式 select column_name from table_name where column_name like 'x%' ##### 5.2.1.1Union联合查询 主要用于把两个查询结果连接起来，放到一个结果集，结果集的列名由第一个select后面的列名决定 条件： 前后两个查询的列名可以不一样，但是列的数量必须一致 基本格式： Select 列 from 表 [where 条件] union 其他查询语句 ##### 5.2.1.2排序 功能： 根据列名来排序，默认按照升序对结果集进行排序，可以使用desx关键字对结果集进行降序排序 格式： Select 列名 from 表名 [where 条件] order by 列名 ##### 5.2.1.3Group by 分组查询，把相同的列放在一起 格式： group by column_name ##### 5.2.1.4子查询 也叫内查询，嵌套查询 是一种嵌套在其他SQL语句中进行的查询 规则： 1.必须在括号内 2.子查询的select语句中，只能由一个列，除非子查询有多个列 select * from ((select 1)A,(select 2)B); 派生表: MySQL在进行子查询时,就会产生一个派生表,他是select语句产生的一个虚拟表,与外部没有关系,产生的结果传给外部select查询 派生表必须定义一个别名，如A，B 使用（）或as定义别名 #### 5.2.2Insert插入语句 插入数据有三种形式： insert into 表名 （列名1，列名2，列名3） values（值1，值2，值3） 省略列名形式： insert into 表名 values（值1，值2，值3） 使用set关键字插入数据 insert into 表名 set 列名1=值1，列名2=值2，列名3=值3 #### 5.2.3Update更新操作 基本格式： Update 表名 set 列名=新值 where 列名=值 更新多个列： Update user set username = ‘tests’ ， password = ‘123’ where id = 3 省略条件，更新表中的一列数据： Update 表名 set 列名=新值 #### 5.2.4Delete删除操作 删除表中的一行数据： Delete from 表明 where 列名=值 使用in关键字删除表中的多行数据 Delete from 表名 where 列名 in （需要删除的行） 删除表中的所有数据 Delete from user ### 5.3DCL语句 ####
