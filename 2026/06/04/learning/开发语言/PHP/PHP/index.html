类型转换是无法避免的,当是两个变量类型不匹配的时候,PHP会自动地进行变量转换。但是PHP是一个弱类型的语言,导致在进行类型转换可能会出现一些问题。 最常见的int与string类型间的转换 int转string可以使用如下方法: $a = 1; $stra=(string)$a; 或 $stra=strval($a); 而string转int可以使用intval函数
