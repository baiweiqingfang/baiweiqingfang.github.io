# 敏感路径和文件汇总 ## 1.信息泄露 ### 1.1PHP探针 PHP 探针 = 把服务器所有敏感配置（OS/PHP/Apache/路径/权限）用网页形式展示出来的脚本，属于信息泄露 常见 PHP 探针文件名： | 文件名 | 说明 | 来源 | | --------------- | ------------------------- | --------------------- | | `/phpinfo.php` | 原生 `phpinfo()` 函数输出 | 开发者调试遗留 | | `/l.php` | phpStudy 默认探针 | phpStudy 一键环境自带 | | `/tz.php` | "探针"拼音缩写 | 常见自定义命名 | | `/p.php` | "PHP"缩写 | 常见自定义命名 | | `/tongji.php` | "统计"拼音 | 常见自定义命名 | | `/info.php` | "信息"缩写 | 常见自定义命名 | | `/test.php` | "测试"文件 | 开发者遗留 | | `/admin888.php` | 伪装成后台 | 恶意后门常见 | #### 1.1.1PHP探针中的敏感信息 1.绝对路径（Absolute Path） 查找文件根目录、DOCUMENT_ROOT或脚本物理路径。 C:/phpStudy/WWW/(Windows)或/var/www/html/(Linux) 2.数据库连接检测 大多数 PHP 探针最下方都有一个“数据库连接检测”模块。 在输入框里尝试常用的弱口令： 主机：127.0.0.1或localhos 用户名：root 密码：root 3.服务器操作系统与环境 服务器操作系统、PHP版本、服务器软件。 4.PHP disable_functions 搜索disable_functions，根据禁用的函数决定绕过方式 5.开启的扩展与组件 GD库、MySQL、Socket、Zend Optimizer ### 1.2robots.txt Robots是一个协议，全称为网络爬虫协议,网站通过Robots协议告诉搜索引擎哪些页面可以访问，哪些不可以访问。 #### 1.2.1User-agent 指定搜索引擎，若为“*”，则表示允许所有搜索引擎 注： （1）当robots.txt不为空的时候，必须至少有一条User-agent的记录 （2）相同名字，只能有一条，但是不同的搜索引擎，可以有多条记录 #### 1.2.2.Disallow 禁止访问的目录 注： （1）在robots.txt中至少要有一条Disallow （2）Disallow记录为空，则表示网站所有页面都允许访问 （3）使用disallow，每个页面必须单独分开声明 #### 1.2.3.Allow 允许访问的目录 #### 1.2.4.Sitemap 向搜索引擎提交网站地图，增加网站收录 ## 2.后台管理 ### 2.1phpMyAdmin /phpmyadmin/README：phpMyAdmin 的官方说明文档，直接暴露版本号，默认位置：phpMyAdmin 安装包根目录 /phpmyadmin/ChangeLog ## 3.配置文件 ### 3.1操作系统 #### 3.1.1Windows C:\Windows\System32\drivers\etc\hosts 显示当前操作系统网卡的IP C:\Windows\win.ini Windows系統配置文件 c\ProgramFiles\mysq/\my.ini MySQL配置 #### 3.1.2Linux /etc/hosts 显示当前操作系统网卡的IP /etc/passwd 账户信息 /etc/shadow 账户密码文件 /proc/net/arp 显示arp缓存表(寻找内网其他主机) /proc/net/fib_trie 显示当前网段路由信息 /proc/net/tcp 记录这台机器当前正在连接的所有内网IP和端口 /var/log/apache2/access.log apache日志文件 /var/log/nginx/access.log nginx日志文件 /var/www/html Apache或Nginx服务器的默认网页根目录 ### 3.2开发语言 #### 3.2.1PHP php.ini 常见的php.ini文件默认路径： phpinfo.php (PHP 配置信息页) DOCUMENT_ROOT/SCRIPT_FILENAME： 告诉了网站在磁盘上的绝对路径 _$SERVER["REMOTE_ADDR"]与内网 IP：观察是否泄露了内网网段信息。 disable_functions：看看服务器是否禁用了命令执行函数 Linux： /etc/php.ini /etc/php/X/apache2/php.ini（将X替换为PHP版本） Windows： C:\php\php.ini C:\Windows\php.ini ##
