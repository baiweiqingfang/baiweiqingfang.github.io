Gopherus

Gopherus是一个专为生成Gopher协议Payload的工具，通过构造特定的Gopher协议数据包，攻击者能够利用SSRF漏洞对内网资源进行访问，甚至可能进一步执行远程代码（Remote Code Execution，RCE）

MySQL：

如果用户没有密码保护，可以导出数据库，甚至在系统中放置恶意文件。

它只会问MySQL用户的用户名，并且会给gopher链接。

PostgreSQL：

如果用户没有密码保护，你可以导出他的数据库，也可以在他的系统中放置恶意文件。

它只要求输入Postgres用户名和数据库名称，然后会给你gopher链接。
