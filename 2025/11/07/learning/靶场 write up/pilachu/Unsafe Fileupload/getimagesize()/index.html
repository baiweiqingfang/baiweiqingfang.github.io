# getimagesize() 1.将一句话木马文件的后缀名改为.jpg，上传显示这是个假图片 2.直接上传图片马 3.利用文件包含漏洞用蚁剑连接 # 注： 1..源代码分析： 文件位置：D:\PHPstudy\phpstudy_pro\WWW\pikachu\vul\unsafeupload|getimagesize.php 发现通过调用upload函数进行验证 upload函数源代码分析： 函数位置:D:\PHPstudy\phpstudy_pro\WWW\pikachu\inc\uploadfunction.php 2.分析代码发现使用了getimagesize()函数进行验证 getimagesize()函数可以判断文件是否为真实的图片，但是只检查文件是否符合图片的基本格式特征，无法检测文件中是否隐藏了恶意代码。攻击者可以通过制作图片马将恶意代码嵌入到正常图片文件的末尾，从而绕过getimagesize()验证 3.文件包含漏洞源代码分析： 文件位置：D:\PHPstudy\phpstudy_pro\WWW\pikachu\vul\fileinclude\fi_local.php 分析代码可知传参点在filename变量，且submit不为空是执行文件包含 4.相对路径，例： 图片木马的URL路径为 http://127.0.0.1/pikachu/vul/unsafeupload/uploads/2024/10/17/45001567107604bd4ee110816937.png 文件包含关卡的URL路径为 http://127.0.0.1/pikachu/vul/fileinclude/fi_local.php?filename=file1.php&submit=1 图片马相对于文件包含关卡的路径为：../../unsafeupload/uploads/2024/10/17/45001567107604bd4ee110816937.png 4.编写的URL：http://127.0.0.1/pikachu/vul/fileinclude/fi_local.php?filename=../../unsafeupload/uploads/2025/11/07/867292690d9c893e41c508193139.jpg&&submit=1
