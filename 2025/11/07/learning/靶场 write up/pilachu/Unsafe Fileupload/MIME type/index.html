MIME type

2025.11.07 5 分钟阅读
# MIME type 1.直接上传.php文件失败 2.bp抓包分析，得到拦截的包，有如下文本： ntent-Disposition: form-data; name="uploadfile"; filename="yjh.php" Content-Type: application/octet-stream 3.将第二行改为Content-Type: image/jpg #白名单中任意一中类型即可 4.放行，得到文件上传位置 5.拼接URL,用蚁剑进行连接 # 注： 1.MIME(Multipurpose Internet Mail Extensions,多用途互联网邮件扩展，又称 Media Type)，content-type的值，常见MIME类型： | 文件类型 | MIME类型 | 典型扩展名 | | ------------ | ----------------------- | -------------- | | 文本文件 | text/plain | .txt | | HTML文件 | text/html | .html | | Markdown文本 | text/markdown | .md, .markdown | | JPEG图片 | image/jpeg | .jpg, .jpeg | | PNG图片 | image/png | .png | | GIF图片 | image/gif | .gif | | PDF文档 | application/pdf | .pdf | | JavaScript | application/javascript | .js | | JSON数据 | application/json | .json | | ZIP压缩文件 | application/zip | .zip | | PHP脚本 | application/x-httpd-php | .php | 2.内容类型（Content-Type），这个标头区域用于指定信息的类型，不仅出现在响应，在上传请求的正文里每一块文件数据前也会出现。 3.服务器接收POST请求后，会验证MIME是否符合白名单,只允许白名单上的文件类型上传 4.源代码： 文件位置：D:\PHPstudy\phpstudy_pro\WWW\pikachu\vul\unsafeupload\servercheck.php upload_sick()函数： 文件位置：D:\PHPstudy\phpstudy_pro\WWW\pikachu\inc\uploadfunction.php
