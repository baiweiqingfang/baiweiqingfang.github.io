命令行里，>> 是一个重定向操作符，作用是把命令的输出追加到指定文件的末尾，而不是覆盖原文件 例： echo "hello" >> file.txt 4.:set shell=/bin/sh 设置内部选项，把选项 shell 改成 /bin/sh :shell打开一个交互式 shell,执行后 vim 会挂起，暂时进入 /bin/sh（或 set shell= 指定的那个解释器）里，在 shell 里 exit 后又回到 vim。 5.Linux命令目录及用途 | 目录 | 用途 | | ----------------- | ----------------------------- | | `/usr/local/bin` | 本地用户程序，供所有用户使用 | | `/usr/local/sbin` | 本地系统管理程序（root 专用） | | `/usr/bin` |
